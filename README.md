## 🎯 Project Overview

### What is NSCS 2026?
The **National Esports Championship Series 2026** is a comprehensive digital experience platform designed for the BPA User Experience Design Team competition. This solution demonstrates professional-grade UX design, technical excellence, and business value for the premier esports championship event in Nashville, TN.

### Key Statistics
- **52 Competing Teams** across 3 games (Valorant, Rocket League, Super Smash Bros)
- **$1M+ Prize Pool** with detailed breakdown
- **Nashville Venue** with accessibility features
- **500,000+ Expected Visitors** to the digital platform

## 🏗️ Architecture

### Technology Stack
- **Frontend**: React 18, Vite, Tailwind CSS
- **Icons**: Lucide React
- **Hosting**: Netlify with global CDN
- **Analytics**: Google Analytics 4

### Project Structure
```
src/
├── components/          # Reusable UI components
│   ├── landing/        # Landing page sections (13 total)
│   ├── Navigation.jsx  # Main navigation
│   └── Footer.jsx      # Site footer
├── data/               # Mock data and constants
├── hooks/              # Custom React hooks
└── App.jsx             # Main application
```

## 🎮 Features

### ✅ BPA Competition Deliverables
- **Promotional Landing Page** - 13 comprehensive sections with interactive features
- **Mobile App Prototype** - 5-screen showcase (Live Scores, Stats, Replays, Brackets, Chat)
- **Social Media Campaign** - Downloadable artboards for Twitch/YouTube Gaming

### 🎯 Interactive Components
- **Countdown Timer** - Real-time countdown to event date
- **Game Showcase** - Interactive game cards with prize pools
- **Team Rosters** - Searchable team database with player information
- **Live Streaming** - Mock streaming interface with controls
- **Interactive Schedule** - Dynamic event timeline with filtering
- **Ticketing System** - Multi-tier pricing with purchase flow
- **Venue Map** - Clickable map with Google Maps integration

### 📱 Responsive Design
- **Mobile-First** - Optimized for all device sizes
- **Touch-Friendly** - Intuitive mobile interactions
- **Fast Loading** - <2 second page load times
