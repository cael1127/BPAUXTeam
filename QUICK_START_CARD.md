## 🚨 TROUBLESHOOTING

**Build fails?**
```bash
rm -rf node_modules
npm install
npm run build
```

**Site not loading?**
- Check URL is correct
- Try incognito mode (no cache)
- Clear browser cache
- Wait 30 seconds for Netlify CDN

**Figma link broken?**
- Share settings: "Anyone can view"
- Copy "Present" link (not "Edit" link)
- Test in incognito browser
